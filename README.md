## Project Setup
#### Create database (MySQL)
- name: ```daftar_belanja```
- table: ```items```

| Fields | Type | Length |
|---|---|---|
| id | integer | 11 
| name | varchar | 30

#### Install dependencies
```
npm install
```

## Build
```
npm run build
```

## Run
```
npm run start
```

Go to http://localhost:3000
